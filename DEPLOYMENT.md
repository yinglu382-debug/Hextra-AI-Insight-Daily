### 停止服务
```bash
# 查找进程
ps aux | grep app.py

# 终止服务
pkill -f app.py
```

### 重启服务
```bash
# 方法1: 使用启动脚本
./start.sh

# 方法2: 直接运行
python3 app.py
```

### 注意事项

1. **首次启动**: 服务器启动后会在后台自动抓取最新数据
2. **数据缓存**: 新闻数据会缓存到 `news_data.json` 文件
3. **日志输出**: 运行时会显示详细的抓取日志
4. **网络要求**: 需要能够访问目标新闻网站

### 技术特点

- **响应式设计**: 支持手机和桌面访问
- **现代化界面**: 使用渐变色和阴影效果
